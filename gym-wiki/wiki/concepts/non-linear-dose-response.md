# Non-Linear Dose Response

## Definition
Training volume follows a diminishing returns curve where each additional set provides progressively smaller contributions to hypertrophy stimulus. The first set provides the majority of the stimulus.

## Research Evidence

### Schoenfeld Metro-Analysis
- 1st set: 100% stimulus
- 2nd set: 39% (major drop-off)
- 3rd set: 22%
- 4th-5th sets: ~13-16% each

### Pelland Metro-Analysis
- 1st set: 100% stimulus
- 2nd set: 82%
- 3rd set: 68%
- 4th set: 57%

## Practical Implications

### Set Priority
- First set is "king" - provides majority of stimulus in any workout
- Second set provides meaningful but reduced benefit
- Third set is optional for those with capacity
- Four or more sets often provide "junk volume" with minimal return

### Paul Anderson Example (1954)
- Trained 6 days/week: 3x upper body + 3x squat-specific days
- Used only 1-3 sets of 2-3 reps per exercise
- Achieved massive leg development squatting 300kg+
- Demonstrates low-volume, high-frequency can build substantial muscle
