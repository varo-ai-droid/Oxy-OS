# Advanced Offer Stacking

## Definition
**Offer Stacking** is the practice of combining multiple value elements into a single cohesive offer to dramatically increase perceived value while maintaining profitability. It involves layering [[Attraction Offers]], [[Upsell Offers]], [[Downsell Offers]], and [[Continuity Offers]] strategically.

## The Value Grid Framework

The Value Grid maps how to stack offers effectively:

### Front End vs Back End
- **Front End:** Initial offer (primary attractor)
- **Back End:** Everything that happens after the initial purchase

### Stacking Considerations
- Each layer should provide genuine value
- Pricing should reflect true costs
- Complexity should remain simple for the customer
- Each offer should flow naturally to the next

## Advanced Stacking Techniques

### 1. Free Presentations
Using educational content as an attraction offer:
- Provides immediate value (knowledge)
- Builds trust and credibility
- Demonstrates expertise
- Creates natural upsell opportunities

### 2. Freemium Model
Partial service offered for free with paid upgrade paths:
- Removes financial risk for initial commitment
- Demonstrates core value
- Creates natural progression to paid tiers

### 3. Pick Your Price
Customer chooses their price point:
- Increases engagement through choice
- Reveals price sensitivity
- Builds trust through flexibility
- Can be combined with [[Decoy Offer]] strategy

## The Stacking Process

### Step 1: Foundation - Your Core Offer
Start with your primary [[Grand Slam Offer]] - what you're fundamentally selling.

### Step 2: Attraction Layer
Add an [[Attraction Offer]] that gets attention:
- Free presentation for knowledge businesses
- Freemium for software/services
- Pick-your-price for flexible pricing acceptance

### Step 3: Upsell Sequence
Design [[Upsell Offers]] that naturally follow:
- Address next logical needs
- Remove additional effort/sacrifice
- Increase transaction value progressively

### Step 4: Continuity Hook
Create [[Continuity Offers]] for ongoing relationship:
- Lifetime upgrades for value retention
- Discounted continuity for long-term commitment
- Combined models for immediate value + ongoing relationship

## Stacking Examples

### Knowledge Business Stack
1. Free presentation (attraction)
2. Paid workshop/course (upsell)
3. Coaching program (continuity)
4. Community membership (ongoing value)

### Product Business Stack
1. Freemium version (attraction)
2. Premium product (upsell)
3. Accessories/add-ons (upsell)
4. Subscription refills (continuity)

## Key Principles

### Each Layer Must Add Value
Don't stack for complexity's sake. Every offer element should solve a genuine problem or fulfill a desire.

### Price According to Perception
Stacked offers can support higher total value because of the cumulative effect of multiple benefits.

### Test and Iterate
Start simple and add layers based on customer response and profitability.
