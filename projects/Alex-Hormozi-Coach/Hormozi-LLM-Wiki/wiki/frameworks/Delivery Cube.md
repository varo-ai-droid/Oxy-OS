# Delivery Cube

## Definition
The **Delivery Cube** is Alex Hormozi's mental model for brainstorming and optimizing how you deliver value to customers. It helps you systematically think through delivery methods to maximize perceived value while minimizing effort.

## The Six Dimensions

### 1. Level of Personal Attention
How much direct access does the customer have to you or your team?

| Level | Description | Use Case |
|-------|-------------|----------|
| **1-to-1** | Individual, personalized attention | High-ticket services, coaching |
| **Small Group** | Limited group (2-20 people) | Workshops, small cohorts |
| **1-to-Many** | One message to many recipients | Courses, group programs |

### 2. Level of Effort Expected from Customer
What does the customer need to do to achieve their outcome?

| Level | Acronym | Description |
|-------|----|-------------|
| **DIY** | Do It Yourself | Customer figures it out on their own |
| **DWY** | Do It With You | You teach/guide them through the process |
| **DFY** | Done For You | You execute the solution for them |

### 3. Delivery Medium
The channel through which value is delivered

- **In-person** - Face-to-face interaction
- **Video** - Recorded or live video content
- **Audio** - Podcasts, calls, voice messages
- **Text** - Written content, email, chat

### 4. Consumption Method
How the customer consumes the offering

- **Live** - Real-time interaction
- **Recorded** - Pre-recorded content
- **On-demand** - Available when they want it
- **Hybrid** - Combination of live and recorded

### 5. Speed & Convenience
Response time and accessibility

| Level | Description |
|-------|-------------|
| **24/7** - Instant response anytime |
| **Business Hours** - Standard response times |
| **Scheduled** - Specific time slots only |
| **Delayed** - Longer response windows acceptable |

### 6. 10x to 1/10th Stretch Test
A powerful brainstorming exercise for offer tier creation

**Questions to ask:**
- If customers paid **10x** the price, what would you provide?
- If customers paid **1/10th** the price, what would you provide?

This helps identify:
- Premium tier offerings (luxury, ultra-high-touch)
- Economy tier offerings (budget-friendly options)
- Natural upsell/downsell paths

## Application in Offer Creation

### Step 1: Brainstorm Across All Dimensions
Use each dimension to generate multiple delivery options for the same outcome.

### Step 2: Map to Value Equation
Choose methods that:
- Increase Dream Outcome
- Increase Likelihood of Achievement
- Decrease Time Delay
- Decrease Effort & Sacrifice

### Step 3: Create Tiered Offerings
Use the Stretch Test to build:
- **Premium** (10x level)
- **Standard** (current offering)
- **Entry** (1/10th level)

## Real-World Examples

### Consulting Business Stack
- **Entry:** DIY spreadsheet template (email delivery)
- **Standard:** Group coaching calls (video, DWY, scheduled)
- **Premium:** Done-for-you strategy + weekly calls (1:1, DFY, 24/7 access)

### Fitness Business Stack
- **Entry:** Recorded workout videos (self-paced, DIY)
- **Standard:** Small group training (live, DWY, 3x/week)
- **Premium:** Personal training + nutrition (1:1, DFY, daily support)

## Key Insights

### Each Dimension Can Be Stretched Independently
You can increase personalization without changing the medium, or increase speed without changing effort level.

### The Sweet Spot for Service Businesses
Often lies in making things **more convenient** rather than **more intensive** - customers value time savings over added complexity.

### Psychology Over Logic
The Delivery Cube is used to create **perceived value**, not just efficient delivery. Think about what feels most valuable to the customer.
