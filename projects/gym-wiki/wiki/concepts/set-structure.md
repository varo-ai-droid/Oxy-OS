# Set Structure

## Definition
The organization of sets and repetitions within a training session, including the number of sets, rep ranges, rest periods, and progression schemes used for each exercise.

## Common Set Structures

### Straight Sets
- Same weight and rep target across all sets
- Simplest and most common approach
- Allows consistent monitoring of progress

### Pyramid Schemes
- **Ascending pyramid**: Increasing weight, decreasing reps (e.g., 10, 8, 6)
- **Descending pyramid**: Decreasing weight, increasing reps (e.g., 6, 8, 10)
- **Full pyramid**: Ascend then descend (e.g., 10, 8, 6, 8, 10)
- Historical precedent in silver era bodybuilding (e.g., 10-8-6 with back-off set of 15)

### Cluster Sets
- Rest periods placed within a set to allow fatigue dissipation
- Enables more total reps at higher loads
- See [[Cluster Sets]] for detailed coverage

### Back-Off Sets
- A final set with reduced weight and higher reps after heavier work
- Adds volume but with diminishing returns on stimulus
- Creates disproportionate fatigue relative to stimulus

## Set Structure Considerations

### Stimulus vs Fatigue
- The first set of an exercise provides the most stimulus
- Each additional set has diminishing returns (non-linear dose response)
- Set structure should balance total stimulus against accumulated fatigue

### Exercise-Specific Considerations
- Multi-joint exercises may benefit from pyramid warm-up structures
- Machine exercises require fewer warm-up sets
- Single-joint exercises can often go straight to working sets
