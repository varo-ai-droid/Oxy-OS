## Your Task
Create a complete, production-ready landing page as a SINGLE HTML file with embedded CSS/JS.

### Required Sections
1. **Hero**: Headline, subhead, primary CTA, social proof bar
2. **Problem**: Visceral pain (use real customer quotes)
3. **Solution**: Product demo (interactive mockup or screenshots)
4. **How It Works**: 3-step visual flow
5. **Pricing**: Simple, transparent, value-anchored
6. **Social Proof**: Testimonials, logos, metrics
7. **Founder Note**: Personal message (from Phase 7)
8. **Footer**: Links, legal, newsletter capture

### Technical Requirements
- Semantic HTML5
- Mobile-first responsive CSS (embedded)
- Vanilla JS only (smooth scroll, FAQ accordion, copy button)
- Zero external dependencies
- WCAG AA contrast
- <100KB total

### Conversion Optimization
- Single primary CTA above fold
- Sticky header with CTA
- Exit-intent newsletter capture
- FAQ addressing top objections

## Output
Return ONLY the complete HTML file. No explanations.
