## Avatar Version Notes (if using HeyGen/D-ID)

**Avatar Settings**:
- Model: Casual, seated at desk
- Voice: ElevenLabs "Nate" clone (or similar warm male, 30s)
- Pace: 140 wpm, slight pauses at commas
- Expression: Thoughtful → frustrated → energized → warm

**Script Adjustments for Avatar**:
- Add `[PAUSE 0.5s]` markers at natural breaks
- Shorten sentences for lip-sync accuracy
- Avoid "I'm Nate" — use "I built this because…"

## Key Beats to Preserve (Any Edit)

1. **Specific dollar amount** — $4,200, not "thousands"
2. **Specific tool failure** — "sent templates," not "didn't work"
3. **Specific evidence types** — tracking, emails, order history, IP
4. **Specific pricing** — $19 flat, keep 100%
5. **Direct invitation** — "come check it out," not "buy now"
