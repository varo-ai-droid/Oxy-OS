## Problem
- **$30B+** annual chargeback losses (e-commerce)

## Solution
**CounterBrief = Evidence automation, not managed service.**

```
Dispute webhook → Parallel fetch (Shopify + Gmail + UPS + ShipStation + ERP)
                → Reason-code mapper (18 codes × 20 evidence types)
                → Packet builder (PDF + structured JSON)
                → Merchant reviews in 2 min → One-click submit
                → Outcome tracked → Win-rate analytics by code
```
