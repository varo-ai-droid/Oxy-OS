Questions

Frequently asked

How is this different from Chargeflow or Chargebacks911?

Those are managed services — they handle disputes FOR you using generic templates. CounterBrief is evidence automation — we build the perfect packet so YOU can review and submit in 2 minutes. Reason-code-native. Transparent. You control the submit.

Result: 55%+ win rate vs their 20-30%. And you pay $19/win, not $200-5000/mo regardless of outcome.

What if I lose the dispute? Do I still pay?

No. You only pay $19 when you approve a response packet. If you don't submit, or if you lose, you pay nothing. We only make money when you recover revenue.

Which reason codes do you support?

All 18 Visa/Mastercard reason codes. The most common for e-commerce:

  • 10.4 Fraud — Card Absent (AVS, CVV, IP, 3DS, tracking)
  • 13.1 Not Received (tracking, delivery proof, signature)
  • 13.2 Cancelled Recurring (cancellation proof, policy, usage logs)
  • 13.3 Not as Described (photos, specs, comms, return tracking)
  • 13.6 Credit Not Processed (refund proof, timeline)

Each code gets its exact evidence requirements — no generic templates.

Do I need technical skills to set this up?

No. Install from Shopify App Store → click "Connect" on each integration (Shopify, Gmail, ShipStation, UPS) → done. 2-click OAuth for each. No code, no webhooks to configure.

If you use email forwarding instead of OAuth, just forward chargeback-related emails to your unique CounterBrief address. Takes 30 seconds.

What data do you access? Is it secure?

We read: order data, fulfillment data, customer timeline, tracking events, email threads (subject/body/sender), carrier delivery scans. We never read: full credit card numbers, passwords, unrelated emails.

SOC 2 Type II certified. GDPR/CCPA compliant. 30-day auto-delete after dispute resolution. PCI SAQ A (we don't touch card data). All data encrypted at rest (AES-256) and in transit (TLS 1.3).

Does this work with Stripe, PayPal, or Braintree disputes?

MVP (launch) supports Shopify Payments + Shopify Payments + Stripe via Shopify. PayPal, Braintree, Adyen, and direct Stripe support are on the roadmap for Q1 2025. Join the waitlist for early access.

Can I use this alongside Chargeflow?

Yes. Many merchants run both for 30-60 days to compare win rates side-by-side. CounterBrief shows you the exact evidence packet before submit — you'll see the difference immediately.

What's the onboarding time?

Core plan: 5 minutes. Install app → connect integrations → first dispute auto-builds.

Pro plan: 15-20 minutes. Adds OAuth email + carrier APIs.
