## What This Skill Does

Sends a daily Telegram message with prioritized news summaries. P1 topics (AI, Business, Tech) get 3 stories each, P2 (World, Germany) get 2 each, P3 (Science) gets 1. Non-English articles are translated.

## Steps

1. If $ARGUMENTS is provided, filter to that topic only. Otherwise run full summary.
2. Load news feed URLs from `context/knowledge/news-feeds.md`.
3. For each topic in priority order (P1→P2→P3):
   - Fetch stories from configured RSS feeds
   - Translate any non-English content using googletrans
   - Summarize to 2 sentences max
4. Format the message with markdown
5. Execute `automations/daily-news/send_news.py` to send via Telegram

## Output Format

```
📰 *DAILY NEWS SUMMARY* - {date}

**P1 HIGH PRIORITY**

**AI (3 stories)**
1. {headline} - {2-sentence summary}
   Source: {source} | [Read more]({link})

...repeat for Business, Technology...

**P2 WORLD & GERMANY**

**World News (2 stories)**
...similar format...

**P3 SCIENCE**

**Science (1 story)**
...single story...

_Generated by Oxy OS daily-news automation_
```

## Configuration Files

- `context/knowledge/news-feeds.md` - RSS feed URLs per topic
- `.env` - Telegram BOT_TOKEN and CHAT_ID (create from .env.example)
- `automations/daily-news/send_news.py` - Python script

## Telegram Setup (One-time)

1. Open Telegram, search `@BotFather`
2. Send `/newbot` and follow prompts
3. BotFather gives you a token - save it
4. Message your bot at least once
5. Visit: `https://api.telegram.org/bot<TOKEN>/getUpdates` (replace TOKEN)
6. Find your chat_id in the response
7. Create `.env` file with:
   ```
   BOT_TOKEN=your_token_here
   CHAT_ID=your_chat_id_here
   ```
